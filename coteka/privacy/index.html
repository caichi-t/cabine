プライバシーポリシー

本ポリシーは、カップル向け立替・精算アプリ「Coteka」における情報の取り扱いについて説明するものです。

制定日: 2026年7月10日 / 最終更新日: 2026年7月10日

日本語 English

1. 対象

本ポリシーは、スマートフォン向けアプリ「Coteka」（以下「本アプリ」）および本ポリシーを掲載する Web ページに適用されます。

2. 取得・保存する情報

本アプリは、ログイン画面なしで利用できるよう、匿名認証により利用者ごとの識別子とセッション情報を作成・保存します。

本アプリの機能提供のため、以下の情報を保存します。

  • グループ情報、招待トークン、メンバーの役割、表示名、手取り額、端数処理、既定の分け方、表示通貨
  • 支払いの内容、金額、日付、支払者、分割方法、各メンバーの負担額
  • 精算の金額、日付、支払者、受取者
  • 定期支払いの内容、金額、支払者、分割方法、発生日、開始月、終了月
  • オンボーディング完了状態、表示設定、認証セッションなど、端末内に保存される本アプリの設定情報

本アプリは、氏名、メールアドレス、電話番号、住所の入力を必須としていません。ただし、利用者が表示名や支払い内容などに個人に関する情報を入力した場合、その内容は本アプリのデータとして保存されます。

3. 利用目的

取得・保存した情報は、以下の目的で利用します。

  • 本アプリの機能提供、グループ作成、招待、同期、立替・精算計算
  • 支払い、精算、定期支払い、メンバー設定、通貨設定の保存と表示
  • 不具合対応、問い合わせ対応、サービス改善
  • 不正利用の防止、セキュリティ確保、法令やストア規約への対応

4. 共有および第三者サービス

同じグループに参加しているメンバーは、グループ内の表示名、手取り額、支払い、精算、定期支払いなどの共有データを閲覧・更新できる場合があります。招待リンクを共有する相手は、信頼できる相手に限定してください。

本アプリは、認証、データベース、リアルタイム同期のために Supabase を使用します。Supabase は、サービス提供、セキュリティ、不正防止、運用、法令遵守のため、アプリデータ、認証情報、端末や通信に関する技術情報を処理する場合があります。

Supabase Privacy Policy

当方は、法令に基づく場合を除き、利用者の情報を第三者へ販売しません。

5. 端末内の保存

本アプリは、認証セッションや設定情報を端末内に保存します。認証セッションは、端末のセキュアな保存領域および暗号化されたローカルストレージを利用して保護されます。

本アプリを削除すると、端末内に保存された本アプリのローカルデータは通常削除されます。ただし、サーバーに保存されたグループデータは、アプリの削除だけでは削除されない場合があります。

6. データの確認・訂正・削除

利用者は、本アプリ内で表示名、手取り額、支払い、精算、定期支払いなどを確認、編集、削除できる場合があります。

退会機能を使用すると、グループから退出できます。相手が参加していない場合は、グループデータが削除されます。相手が参加している場合は、相手側で共有履歴を継続して利用できるよう、過去の支払い・精算などの共有データが残る場合があります。

上記以外の情報の確認、訂正、削除をご希望の場合は、本ポリシー末尾の連絡先までお問い合わせください。本人確認や技術上・法令上の制約により、すべてのご要望に対応できない場合があります。

7. 未成年者の利用

未成年者が本アプリを利用する場合は、保護者の同意と管理のもとで利用してください。保護者から情報の取り扱いに関するお問い合わせがあった場合、合理的な範囲で対応します。

8. 安全管理

当方は、本アプリで取り扱う情報について、紛失、漏えい、改ざん、不正アクセスなどを防ぐため、合理的な安全管理措置を講じるよう努めます。

9. 本 Web ページについて

本 Web ページは GitHub Pages によりホスティングされています。当方は、本ページに独自のアクセス解析ツール、広告タグ、Cookie を設置していません。GitHub による情報の取り扱いについては、GitHub のプライバシー声明をご確認ください。

GitHub General Privacy Statement

10. 変更

当方は、必要に応じて本ポリシーを変更することがあります。重要な変更がある場合は、本ページ上で告知します。変更後のポリシーは、本ページに掲載された時点から効力を生じます。
