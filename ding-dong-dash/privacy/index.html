プライバシーポリシー

本ポリシーは、ゲームアプリ「Ding Dong Dash」における情報の取り扱いについて説明するものです。

制定日: 2026年7月10日 / 最終更新日: 2026年7月10日

日本語 English

1. 対象

本ポリシーは、スマートフォン向けゲームアプリ「Ding Dong Dash」（以下「本アプリ」）および本ポリシーを掲載する Web ページに適用されます。

2. 当方が直接取得する情報

本アプリは、ユーザー登録、氏名、メールアドレス、電話番号、住所など、利用者を直接特定する情報の入力を求めません。

本アプリは、ゲームの進行状況、クリア状況、選択した言語、チュートリアル完了状態、広告表示に関する内部カウントを、利用者の端末内に保存します。これらの情報は本アプリの機能提供のために使用され、当方のサーバーへ送信されません。

3. 広告配信および第三者サービス

本アプリは広告配信のために Unity Ads を使用します。Unity Ads は、広告の表示、広告品質の維持、不正防止、効果測定、法令遵守、利用者の選択に応じた広告配信のため、端末情報、IP アドレス、広告識別子、アプリ利用状況、広告の表示・クリック・完了状況などを取得する場合があります。

Unity による情報の取り扱いについては、Unity の「Game Player and App User Privacy Policy」をご確認ください。

Unity Game Player and App User Privacy Policy

4. 利用目的

当方および第三者サービスは、取得した情報を以下の目的で利用します。

  • 本アプリの機能提供、進行状況の保存、表示言語の保持
  • 広告の表示、広告在庫の管理、広告品質の維持
  • 不正行為や不正な広告表示の防止
  • 不具合対応、サービス改善、問い合わせ対応
  • 法令、ストア規約、広告ネットワーク規約への対応

5. 第三者提供

当方は、法令に基づく場合を除き、利用者を直接特定する情報を第三者へ販売または提供しません。ただし、本アプリに組み込まれた第三者サービスは、それぞれのプライバシーポリシーに従って情報を処理する場合があります。

6. 利用者の選択

利用者は、端末の設定から広告識別子のリセット、パーソナライズ広告の制限、アプリの権限設定の変更を行える場合があります。設定項目の名称や利用可否は、OS、端末、地域、ストアの仕様により異なります。

本アプリを削除すると、端末内に保存された本アプリのローカルデータは通常削除されます。

7. 未成年者の利用

本アプリは、利用者から年齢、氏名、連絡先などを直接収集しません。未成年者が本アプリを利用する場合は、保護者の同意と管理のもとで利用してください。保護者から情報の取り扱いに関するお問い合わせがあった場合、合理的な範囲で対応します。

8. 安全管理

当方は、本アプリで取り扱う情報について、紛失、漏えい、改ざん、不正アクセスなどを防ぐため、合理的な安全管理措置を講じるよう努めます。

9. 本 Web ページについて

本 Web ページは GitHub Pages によりホスティングされています。当方は、本ページに独自のアクセス解析ツール、広告タグ、Cookie を設置していません。GitHub による情報の取り扱いについては、GitHub のプライバシー声明をご確認ください。

GitHub General Privacy Statement

10. 変更

当方は、必要に応じて本ポリシーを変更することがあります。重要な変更がある場合は、本ページ上で告知します。変更後のポリシーは、本ページに掲載された時点から効力を生じます。
